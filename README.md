# Frivillig Opgavestyring & Kalender

Dette er backend og frontend til Frivillig-appen.

## Funktioner
- Opgavestyring for frivillige
- Kalenderfunktion
- Rolle-baseret adgang (SQL)

## Teknologier
- TypeScript
- Vite
- SQL (Database setup)

## Installation
1. Kør `npm install` for at hente afhængigheder.
2. Opsæt databasen med `setup.sql`.
